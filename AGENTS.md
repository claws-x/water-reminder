# Mobile App Project – Agent Team Overview

本项目是一个面向 iOS / Android 的移动应用孵化平台，由一支虚拟的顶级产品与工程团队共同完成。

## 核心角色列表

| 角色 | Agent ID | 职责 |
|------|----------|------|
| `pm` | pm | 首席产品经理 |
| `ux` | ux | 体验与界面设计师 |
| `ios` | ios | iOS 客户端工程师 |
| `qa` | qa | 测试与质量保障工程师 |
| `ops` | ops | 发布与运营支持 |

## 当前项目：HydroTrack

| 角色 | 状态 | 任务 |
|------|------|------|
| pm | ✅ 完成 | 需求文档 |
| ux | 🟡 进行中 | 界面设计 |
| ios | 🟡 进行中 | 代码开发 |
| qa | ⏳ 待开始 | 测试计划 |
| ops | ⏳ 待开始 | 发布准备 |

## 团队协作流程

```
1. pm 定义需求 → docs/requirements/
2. ux 设计界面 → docs/architecture/
3. ios 实现代码 → src/ios/
4. qa 编写测试 → docs/test-plans/
5. ops 准备发布 → docs/process-reports/
```
