## 📱 产品介绍

**HydroTrack** 是一款精美的喝水追踪 + 定时提醒应用，帮助用户保持充足水分，养成健康饮水习惯。

### 核心功能

- 💧 **喝水记录** - 一键记录，支持多种容器
- ⏰ **智能提醒** - 定时提醒，避免忘记喝水
- 📊 **数据统计** - 每日/每周/每月趋势
- 🏆 **成就系统** - 激励坚持好习惯
- 🎨 **精美界面** - 支持深色模式 + 动画

## 🚀 快速开始

### 环境要求

- macOS 12.0+
- Xcode 14.0+
- iOS 15.0+

### 构建项目

```bash
# 打开工程
open src/ios/HydroTrack.xcodeproj

# 或使用命令行编译
xcodebuild -scheme HydroTrack \
           -destination 'platform=iOS Simulator,name=iPhone 15' \
           clean build
```
