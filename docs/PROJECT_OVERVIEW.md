## 📱 产品简介

**HydroTrack** 是一款精美的喝水追踪 + 定时提醒应用，帮助用户保持充足水分，养成健康饮水习惯。

### 核心特色

- 💧 **喝水记录** - 一键记录，支持多种容器
- ⏰ **智能提醒** - 定时提醒，避免忘记喝水
- 📊 **数据统计** - 每日/每周/每月趋势
- 🏆 **成就系统** - 激励坚持好习惯
- 🎨 **精美界面** - 支持深色模式 + 动画
