## 🚀 提交流程

### 1. 准备工作

- [ ] 创建 Apple Developer 账号 ($99/年)
- [ ] 准备所有材料
- [ ] 测试最终版本

### 2. 创建 App Record

- [ ] 登录 App Store Connect
- [ ] 点击 "+" 创建新 App
- [ ] 选择分类：Health & Fitness
- [ ] 填写应用信息
- [ ] 上传截图和图标

### 3. 构建 Archive

```bash
# 在 Xcode 中:
# 1. 选择 "Any iOS Device (arm64)"
# 2. Product → Archive
# 3. 等待归档完成
# 4. 点击 "Distribute App"
# 5. 选择 "App Store Connect"
# 6. 选择 "Upload"
# 7. 按照向导完成
```

### 4. 提交审核

- [ ] 在 App Store Connect 选择构建
- [ ] 填写审核信息
- [ ] 提交审核

### 5. 等待审核

- 通常 1-3 个工作日
- 可在 App Store Connect 查看状态

### 6. 发布上线

- 审核通过后手动或自动发布
